# Weather Knowledge Graph

A mini project demonstrating a **Neo4j knowledge graph** for weather analytics.

## Features
- Models cities, weather, dates, and impacts
- Inserts bulk weather data programmatically
- Queries for:
  - Weather causing flight delays
  - Cities with most storms
  - Weather history for a city
- Supports graph analytics for impact and pattern detection

## Tech Stack
- Neo4j Aura (cloud)
- Python 3
- Google Colab
- Pandas
